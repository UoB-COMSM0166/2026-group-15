# Spray Fun - Week 2 Workshop Homework

**Game Link:** https://editor.p5js.org/Bella-Z/full/zWE4P22ni

## Description
This is our Group 15 submission for the Week 2 Workshop assignment. Spray Fun is an interactive painting game built using p5.js where players can create colorful spray paint art on a digital canvas. The game features dynamic spray effects that respond to mouse movements, allowing users to express their creativity through digital graffiti.

## Features
- Mouse-controlled painting interface
- Colorful visual effects

## Technologies Used
- JavaScript
- p5.js library
